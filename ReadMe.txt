To play the game:
1- open the "Umbra" Folder
2- Then open the "src" Folder -- all our files are placed in there respective locations html,css,js files in src folder. images in the img folder, audio files in the aud folder
3- To Start the game either Open "MainMenu.html" for the English Loading or "Menu.html" for the Spanish Loading.
4- From the Menu Screen before clicking the Start button go into the Settings Screen to set your background audio and your effects audio. They are set to on by default also you can change your language in this menu if you want to try it in another language. 
5. Controls are in the Control Screen W to Jump A to go Left, D to Go Right, L to Punch. A to Continue on the Text. Space Bar to Continue once Level is Beaten.
6.Enjoy!
